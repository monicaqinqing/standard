Credits 鸣谢

对于 Amaze UI，我们不是创造者。

Amaze UI 的诞生，依托于 GitHub 及其他技术社区上一些优秀的资源；Amaze UI 的成长，则离不开用户的支持。

开源不仅是分享，也是协作，通过社区的力量解决问题，促进发展。期待你的参与！

感谢开源！感谢有你！
